# One Health Platform Prototype

This prototype demonstrates a Community Focused Project idea for a unified healthcare information platform.

The platform helps the public, patients, senior citizens, and emergency users quickly find nearby hospitals, doctor availability, bed availability, medicines, pharmacies, ambulance services, insurance support, doctor specializations, health suggestions, and hospital locations.

## Prototype Coverage

- Nearby hospital discovery with beds, ICU count, oxygen support, insurance status, distance, and contact actions
- Doctor specialization and availability listing
- Medicine stock and nearby pharmacy lookup
- Ambulance service listing with ETA and contact actions
- Insurance support workflow mockup
- Map-style service location preview
- AI-style symptom suggestion panel using rule-based triage logic
- Responsive UI using the requested colors:
  - `#003366`
  - `#005B96`
  - `#008CBA`
  - `#A8D5BA`
  - `#F2F2F2`
  - `#333333`
  - `#FF6F61`
  - `#7CA0A1`
